# Giacomini Percezione - V2 (copia indipendente)

Dashboard full-width che NON tocca il progetto originale.

## Deploy su Railway (2 minuti)
1. Crea nuovo progetto su railway.app
2. "Deploy from GitHub" oppure "Empty project" > carica questa cartella
3. Variabili (opzionali):
   - BRANDWATCH_API=https://giacominipercezione-production.up.railway.app/api/brandwatch
   - PORT=3000
4. Deploy

La dashboard chiama automaticamente i dati reali dal tuo backend esistente.
Puoi collegarla a un'altra dashboard senza modificare la repo originale.
